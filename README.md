# Analytics Work

This repository contains a set of analytics projects that showcase different parts of my experience through both Python-based analysis and dashboard reporting.

## Overview

These projects were chosen to show a range of analytics skills across statistical modeling, data analysis, visualization, dashboard design, and reporting.

Rather than including every analytics assignment, this repository focuses on a few representative examples that best reflect my background and range in analytics work.

## Featured Projects

### Forest Fire Regression Analysis
A Python project that uses linear regression to analyze how weather-related variables such as temperature, humidity, wind, and rain relate to forest fire area. The project builds separate models for each variable, calculates summary statistics, generates predictions, and visualizes the regression results.

**Skills demonstrated:**
- working with structured datasets in Python
- simple linear regression
- interpreting slope, intercept, p-value, standard error, and r-squared
- generating predictions from a model

### CS2 Market Analysis Dashboard
A Power BI project focused on analyzing the Counter-Strike 2 market using October 2025 data. The dashboard explores item pricing, sales volume, category performance, and item-level behavior through overview reporting, time-based trend analysis, category drilldowns, and detailed item views.

**Skills demonstrated:**
- Power BI dashboard development
- time-series and trend analysis
- category and item-level reporting
- pricing and volume analysis
- interactive dashboard filtering
- presenting complex data in a clear, usable format
